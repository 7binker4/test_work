На выполнение данного задания потребовалось около 9 часов
Тестовое задание "Добавление/удаление пользователей из таблицы"
Создавалось в Visual Studio Code с использованием OpenServer
В папке "DB" находится база данных с тестовыми пользователями

В папке "php" находится весь функционал
    "Conection" подключение базы данных
    "DeleteUser" - Удаление пользователя
    "EditUser" - Добавление пользователя
    "FormEdit" - Форма добавления пользователя
    "FormInsert" - Форма изменения данных о пользователе
    "Index" - Главная страница
    "InsertUser" - Изменение данных пользователя
    "SearchUser" - поиск пользователей в таблице
    
В папке "ScreenShot" предоставлены скриншоты задания (на всякий случай)
